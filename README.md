jRTClamScan
============

OBD-II Java API

## Build ##

### Requisites ###

* JDK 8
* Maven 3.1 or newer

### Compile, package and install locally ###

```
mvn clean install
```

## Usage ##

java -jar jRTClamScan-x.y.z-SNAPSHOT.jar -Dlog4j.configurationFile=log4j2.xml jRTClamScan.xml
